# 대학영어 영상 플레이어

2020 학년도 당시 대학영어 플레이어가 Internet Explorer 환경에서만 동작함으로 인해

대중화된 Chrome 및 기타 플레이어로 작동할 수 없는 문제가 발생.

때문에 본인과 같이 수강에 차질을 겪는 새내기 학우들을 위하여 제작 후 배포함.
